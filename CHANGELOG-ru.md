# Журнал изменений

Все заметные изменения проекта будут документироваться в этом файле.

## [1.1.0] - 2026-06-19

### Добавлено

- Верхняя btop-style строка статуса в `abtop-py` с CPU, памятью, load average, суммарной памятью агентов, средним использованием контекста, временем и счетчиками сессий.
- Метка ожидания Codex между шагами, чтобы не показывать `waiting for input` во время коротких idle-пауз после сообщений ассистента.

## [1.0.0] - 2026-06-18

### Добавлено

- Первый публичный релиз `abtop-py`.
- Однофайловый Python-монитор без внешних зависимостей для сессий Claude Code и Codex CLI/Desktop.
- Интерактивный curses-дашборд с панелями контекста, квот, токенов, проектов, портов, MCP-серверов, сессий, чата, subagents и timeline.
- Одноразовый текстовый снимок через `--once`.
- JSON-снимок через `--json`.
- Установка Claude StatusLine helper через `--setup`.
- Полностью локальный сбор данных из состояния процессов, файлов сессий, транскриптов, rollout-логов, состояния Git и файлов кэша лимитов.
- Различение ожидания обычного ввода пользователя и явного решения или подтверждения.
- README-файлы на английском и русском языках.
- Архитектурные диаграммы PlantUML на английском и русском языках.
